# 🎓 Online Course Registration Portal

## 📖 Project Description

The **Online Course Registration Portal** is a responsive web application developed using **HTML5, CSS3, and Bootstrap 5**. It enables students to explore available courses, register online, learn about the institute, and contact the training center through a modern and user-friendly interface.

This project is designed for educational institutions and training centers to provide course information and simplify the student registration process.

---
