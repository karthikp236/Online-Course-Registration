🎓 Online Course Registration Portal
📖 Project Description

A responsive Online Course Registration Portal built using HTML5, CSS3, and Bootstrap 5. This web application enables students to explore available courses, register online, learn about the institute, and contact the training center through a modern and user-friendly interface.
