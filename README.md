🎓 Online Course Registration Portal

A responsive web application developed using HTML5, CSS3, and Bootstrap 5 for a training institute where students can explore courses and register online.

📖 Project Description

The Online Course Registration Portal is designed to help students view available courses, learn about the institute, and register for training programs through a user-friendly interface.
The website is fully responsive and provides a professional design suitable for educational institutions.
