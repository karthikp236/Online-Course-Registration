🎓 Online Course Registration Portal

A responsive and user-friendly web application developed using HTML5, CSS3, and Bootstrap 5. This project allows students to explore available courses, register online, learn about the institute, and contact the training center easily.

📖 Project Overview

The Online Course Registration Portal is designed for a training institute to manage course information and student registrations through a modern web interface.

The website consists of multiple pages including:

Home Page
Courses Page
Registration Page
About Us Page
Contact Us Page
